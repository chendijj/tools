# 功能单据分类脚本 - TODO任务清单

## 项目目标
创建一个Python脚本，用于处理功能单据文本，按类别分类并格式化输出。

## 功能需求
1. 读取本地txt文本文件
2. 识别格式：`【类别】功能名称`
3. 按类别分类功能单据
4. 输出格式：`类别及其子功能单据X个`
5. 同类别功能按顺序编号
6. 输出到同目录下的新txt文件
